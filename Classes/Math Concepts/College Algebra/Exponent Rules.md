#mathConcepts #Pre-Calculus 
1. The Product Rule

# $x^n * x^m = x^{n+m}$

2. The Quotient Rule

# $\frac{x^n}{x^m} = x^{n-m}$

3. The Power Rule

# $(x^n)^m = x^{n*m}$

4. Power of Zero

# $x^0 = 1$

5. Negative Exponents

# $x^{-n} = \frac{1}{x^n}$

6. Fractional Exponents

# $x^\frac{1}{n} = \sqrt[n]{x}$

7. Distribute an Exponent over a Product

# $(x*y)^n = x^n * y^n$

8. Distribute an Exponent over a Quotient

# $(\frac{x}{y})^n = \frac{x^n}{y^n}$

### (!) Cannot distribute an exponent over add or subt, only mult + division

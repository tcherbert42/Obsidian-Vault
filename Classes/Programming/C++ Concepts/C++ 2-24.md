# Function Prototype
```c++
// function prototypes are neccessary when the programmer defined function has to go underneath the main function. The main function should always be at the top, so most of the time, the programmer defined functions have to be underneath, warranting the use of a FUNCTION PROTOTYPE

void displayMessage(); //function prototype

int main() {
	cout << "Main Function";
	displayMessage(); //function call
	cout << "Back in main function";
}

void displayMessage() {
	cout << "Hello! I am in displayMessage!"
}

```

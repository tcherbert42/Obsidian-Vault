# Nested Structures
```C++
struct PersonInfo   
{
	string name, adress, city;
};

struct Student
{
    int studentID;
    PersonInfo pData;
    short yearInSchool;
    double dpa;
}; // for exmaple, getline(cin, s.pData.name);
```

- Use the dot operator muliple times to refer to fields of nested structures:

```C++
Student s;
s.pData.name = "Joanne";
s.pData.city = "Tulsa";
```
